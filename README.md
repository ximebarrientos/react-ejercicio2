# Ejercicio 2 - Hola amigo mio

## Descripción

El objetivo es crear una **aplicación web simple** utilizando React que muestre en pantalla el mensaje: Hola 
y enviar mediante props el valor "amigo mio"

## Tecnologías utilizadas

- **React** (Vite)
- **JavaScript**
- **HTML/CSS**
